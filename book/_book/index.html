gitbook简易教程

确保电脑上安装好了nodejs及npm

命令行安装gitbook npm install -g gitbook-cli

根据SUMMARY.md生成对应的章节文件 gitbook init

编辑对应章节的md文件(推荐sublime_text编辑器或是vim)，可能你需要一点Markdown 的基础知识http://www.jianshu.com/p/q81RER

编写好之后就可以编译并启动预览服务器 gitbook serve

编译成功后会监听本地4000目录

浏览器打开 http://localhost:4000查看效果
